# Description
Ingesting tweets using Flume and then counting the number of tweets for each hashtag in hive using a JSON parser SerDe.

## Prerequisites
To run this software you need the following:
1. Linux 
2. Hadoop 2.0
3. Hive 2.0 
4. Flume
5. Twitter Developer App Credentials
